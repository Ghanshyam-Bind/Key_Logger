# Key_Logger
# Basic Python Keylogger

This project is a **simple keylogger built in Python** that records keyboard keystrokes and saves them into a text file.
It demonstrates how keystroke logging works for **learning and cybersecurity awareness purposes**.

---
## ⚠️ Disclaimer

This project is created **for educational purposes only** to understand how keyloggers work in cybersecurity and malware analysis.
It should only be used on systems where you have **explicit permission**.

---
## 🚀 Features

* Captures keyboard input
* Logs **letters and numbers**
* Handles special keys like:

  * **Space**
  * **Enter**
  * **Backspace**
* Saves all keystrokes into a log file
* Stops when **ESC key** is pressed

---

## 🛠 Requirements

* Python 3
* `pynput` library

Install the required library:

```bash
pip install pynput
```

---

## ▶️ How to Run

Run the program using: https://github.com/Ghanshyam-Bind/Key_Logger.git

```bash
python task-4.py
```

Once the program starts, it will begin logging keystrokes into **keylog.txt**.

Press **ESC** to stop the program.
